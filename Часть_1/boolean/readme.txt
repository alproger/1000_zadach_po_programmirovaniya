###Логические выражения

Во всех заданиях данной группы требуется вывести логическое значение TRUE , если приведенное высказывание для предложенных исходных данных
является истинным, и значение FALSE в противном случае. Все числа, для которых указано количество цифр (двузначное число, трехзначное число и т. д.),
считаются целыми положительными.

#Boolean5 ◦ . Даны два целых числа: A, B. Проверить истинность высказывания:«Справедливы неравенства A ≥ 0 или B < −2».

#Boolean6 ◦ . Даны три целых числа: A, B, C. Проверить истинность высказывания: «Справедливо двойное неравенство 
A < B < C» Boolean7 ◦ . Даны три целых числа: A, B, C. Проверить истинность высказывания: «Число B находится между числами A и C».
